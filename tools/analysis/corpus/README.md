# FSM corpus (runtime census dumps)

Generated by the bridge's `/fsmcensus` endpoint (see build-plan.md section 5):
the game's own deserializer is the only faithful FSM parser, so structure is
dumped from the running game — states, actions with parameters, transitions,
events, variable schemas. One JSON line per FSM.

- `fsm-census-title.jsonl` — title scene (level0)
- `fsm-census.jsonl` — station scene (level1), the whole game

Untracked in git (regenerable; re-dump after any game patch — takes seconds).

**SPOILER WARNING (owner: do not open these files):** action parameters embed
narrative strings — outcome texts, dialogue references, unseen content. These
files are analysis inputs only. Reports derived from them use structural
identifiers exclusively, per the rendered-only speech rule.
